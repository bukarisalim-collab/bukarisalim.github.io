## 🚀 Featured Project: DNN Multimodal Architecture

An advanced multimodal deep learning architecture featuring a four-stage pipeline designed for joint text and image processing, analysis, and alignment.

### 📋 Overview
This project implements and evaluates various fusion strategies—including early fusion, late fusion, Graph Neural Networks (GNNs), and CLIP embeddings—to classify multimodal data. It includes fully automated hyperparameter tuning, semi-supervised consistency training with pseudo-labeling, and self-supervised contrastive pretraining.

### 🛠️ Tech Stack & Tools
* **Core Language:** Python (98%)
* **Frameworks:** PyTorch, Torchvision, Transformers (CLIP)
* **Architectures:** Deep Neural Networks (DNN), Graph Neural Networks (GNN)
* **Execution Environment:** Bash/Shell Scripting

### 🌟 Project Stages
1. **Part A (Multimodal Embeddings):** Builds text and image embeddings using baseline networks or CLIP.
2. **Part B (Fusion Comparison):** Compares early fusion, late fusion, GNN early/late variants, and standalone CLIP.
3. **Part C (Semi-Supervised):** Integrates pseudo-labeling and consistency training loops for low-data scenarios.
4. **Part D (Self-Supervised):** Implements contrastive pretraining and downstream transfer learning.

### 💻 Quick Run Commands
To run a quick single-epoch test with the CLIP backbone:
```bash
pip install -r requirements.txt
python part_a_multimodal_embeddings.py --backbone clip --max-samples 256 --epochs 1
```
